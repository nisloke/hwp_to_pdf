# HWP to PDF 변환 프로그램

이 프로젝트는 Gemini CLI와 함께 아래아한글 파일(.hwp, .hwpx)을 PDF 파일로 변환하는 Windows용 데스크톱 애플리케이션을 개발하는 것을 목표로 합니다.

## 주요 기능

1.  **드래그 앤 드롭 변환**: 사용자가 하나 이상의 아래아한글 파일을 애플리케이션 창으로 드래그 앤 드롭하면 자동으로 PDF로 변환됩니다.
2.  **폴더 기반 변환**: 사용자가 특정 폴더를 선택하면, 해당 폴더와 모든 하위 폴더에 있는 아래아한글 파일을 검색하여 목록으로 보여줍니다. 사용자는 목록에서 원하는 파일을 선택하여 한 번에 PDF로 변환할 수 있습니다.
3.  **Windows 탐색기 연동**: Windows 파일 탐색기에서 아래아한글 파일을 마우스 오른쪽 버튼으로 클릭했을 때 나타나는 상황에 맞는 메뉴에 "PDF로 변환" 옵션을 추가합니다.

## 기술 스택

*   **언어**: Python
*   **핵심 로직**: `pyhwp` 패키지를 사용하여 HWP/HWPX 파일을 분석하고, COM 자동화를 통해 설치된 한컴오피스 소프트웨어를 제어하여 PDF로 변환합니다.
*   **GUI**: `Tkinter` 또는 `customtkinter`와 같은 Python GUI 라이브러리를 사용하여 사용자 인터페이스를 구축합니다.
*   **패키징**: `PyInstaller`를 사용하여 모든 종속성과 함께 단일 실행 파일(.exe)로 패키징하여 배포를 용이하게 합니다.
*   **Windows 통합**: Python의 `winreg` 모듈 또는 `.reg` 파일 스크립트를 사용하여 Windows 레지스트리를 수정하고 파일 탐색기 상황에 맞는 메뉴를 추가합니다.
