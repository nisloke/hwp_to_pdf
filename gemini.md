# HWP to PDF 변환 프로그램

이 프로젝트는 Gemini CLI와 함께 아래아한글 파일(.hwp, .hwpx)을 PDF 파일로 변환하는 Windows용 데스크톱 애플리케이션을 개발하는 것을 목표로 합니다.

## 주요 기능

1.  **EXE 파일에 드래그 앤 드롭**: 사용자가 하나 이상의 아래아한글 파일을 애플리케이션 실행 파일(.exe)로 드래그하면 자동으로 실행되어 PDF로 변환됩니다. (가장 편리한 방법)
2.  **GUI 버튼 조작**: 애플리케이션 실행 후 "파일 추가" 또는 "폴더 추가" 버튼을 통해 변환할 파일을 선택할 수 있습니다.
3.  **폴더 기반 자동 검색**: 사용자가 특정 폴더를 선택하면, 해당 폴더와 모든 하위 폴더에 있는 아래아한글 파일을 검색하여 목록으로 보여줍니다.
4.  **Windows 탐색기 연동**: Windows 파일 탐색기에서 아래아한글 파일을 마우스 오른쪽 버튼으로 클릭했을 때 나타나는 상황에 맞는 메뉴에 "PDF로 변환" 옵션을 추가합니다. (`install_context_menu.reg` 실행 필요)
5.  **상세 목록 및 관리**: 파일명, 크기, 수정일, 경로 등 상세 정보를 Treeview로 제공하며, 열 너비 자동 조절 및 정렬 기능을 지원합니다.
6.  **결과 관리**: 변환된 PDF 파일 목록을 별도로 보여주며, 바로 열거나 폴더 열기, 이름 변경, 삭제 등의 편의 기능을 제공합니다.

## 기술 스택

*   **언어**: Python
*   **핵심 로직**: `pyhwp` 패키지를 사용하여 HWP/HWPX 파일을 분석하고, COM 자동화를 통해 설치된 한컴오피스 소프트웨어를 제어하여 PDF로 변환합니다.
*   **GUI**: `customtkinter`와 `tkinter.ttk.Treeview`를 사용하여 현대적이고 직관적인 사용자 인터페이스를 구축했습니다.
*   **패키징**: `PyInstaller`를 사용하여 모든 종속성과 함께 단일 실행 파일(.exe)로 패키징하여 배포를 용이하게 합니다.
*   **자동화**: `pywinauto`를 사용하여 한컴오피스의 보안 경고창("접근 허용")을 자동으로 처리합니다.

## 개발 지침 (주의 사항)

*   **빌드**: `PyInstaller`를 사용하여 실행 파일을 빌드할 때는 **반드시 가상 환경(`venv`) 내의 `pyinstaller` 경로를 명시적으로 사용**해야 합니다.
    *   예: `.\venv\Scripts\pyinstaller.exe --onefile --windowed --name HwpToPdfConverter main.py`
*   **자동화 금지**: 사용자가 명시적으로 요청하기 전까지는 자동으로 빌드하거나 `git push` 등의 배포 작업을 수행하지 마십시오.
