# HWP to PDF 변환기

한컴오피스 아래아한글 파일(.hwp, .hwpx)을 PDF로 변환하는 Windows용 프로그램입니다.

## 주요 기능

*   **드래그 앤 드롭**: 변환하고 싶은 hwp/hwpx 파일을 프로그램 창 안으로 끌어다 놓아 목록에 추가할 수 있습니다.
*   **폴더 선택**: '폴더 선택' 버튼을 눌러 특정 폴더를 지정하면, 해당 폴더와 그 안의 모든 하위 폴더를 검색하여 관련 파일을 자동으로 목록에 추가합니다.
*   **우클릭 메뉴 지원**: Windows 파일 탐색기에서 hwp/hwpx 파일을 직접 우클릭하여 PDF로 변환할 수 있습니다.

## 요구 사항

*   **Windows 운영체제**: 이 프로그램은 Windows에서만 동작합니다.
*   **한컴오피스 한글 설치**: **반드시** 한컴오피스 한글 프로그램이 컴퓨터에 설치되어 있어야 합니다. 이 프로그램은 한글의 자동화 기능을 이용하여 변환을 수행합니다.

## 사용 방법

1.  `dist` 폴더 안에 있는 `HwpToPdfConverter.exe` 파일을 실행합니다.
2.  변환할 파일을 창으로 드래그 앤 드롭하거나, '폴더 선택' 버튼으로 파일을 목록에 추가합니다.
3.  목록에서 변환을 원하는 파일들이 체크되었는지 확인합니다.
4.  '선택 파일 변환' 버튼을 누릅니다.
5.  변환이 진행되며, 완료되면 상태 메시지가 나타납니다. PDF 파일은 원본 hwp/hwpx 파일과 동일한 폴더에 생성됩니다.

## 우클릭 메뉴 설정 방법

파일 탐색기에서 hwp/hwpx 파일을 우클릭하여 바로 PDF로 변환하는 기능을 사용하려면 아래의 설정을 따라주세요.

1.  **경로 수정**: `install_context_menu.reg` 파일을 메모장이나 다른 텍스트 편집기로 엽니다.
2.  아래와 같이 생긴 줄을 찾습니다.
    ```
    @=""C:\\path\\to\\your\\app.exe" "%1""
    ```
3.  `C:\path\to\your\app.exe` 부분을 실제 `HwpToPdfConverter.exe` 파일이 있는 **전체 경로**로 수정합니다. 경로의 각 백슬래시(`\`)는 두 개(`\\`)로 작성해야 합니다.
    *   **예시**: 만약 `D:\Tools` 폴더에 프로그램을 두었다면, `D:\\Tools\\HwpToPdfConverter.exe` 로 수정합니다.
4.  파일을 저장한 뒤, `install_context_menu.reg` 파일을 더블클릭하여 실행하고 레지스트리 추가를 승인합니다.
5.  이제 hwp/hwpx 파일을 우클릭하면 "PDF로 변환" 메뉴가 나타납니다.
